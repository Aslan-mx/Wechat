
## 功能介绍

### 1. 大厂作息时间 小程序

- 你所在的城市有哪些大厂？
- 这些大厂的上下班时间是否弹性的？
- 平时有哪些福利、下午茶？
- 是否写周报、日报？工作氛围怎么样？
- 加班是否有补贴？
- 大厂作息时间 小程序可以帮你解决这些问题。
